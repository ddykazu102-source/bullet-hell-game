# bullet-hell-game
## 概要
JavaScript（Canvas API）を使用して制作した弾幕シューティングゲームです。  
ゲームロジックの設計、状態管理、難易度調整などを自力で実装しました。

Web制作を学習する中で「動きのあるUI」「ロジック設計力」を高める目的で制作しました。

---

## 主な機能

- 難易度選択（弾数・速度・サイズを動的変更）
- ボムシステム
- ボス撃破時の残機回復
- リプレイ機能
- スマホタッチ操作対応
- UI表示（スコア・残機・ボム数）

---

## 使用技術

- HTML5
- CSS3
- JavaScript（ES6）
- Canvas API

---

## 工夫した点

### ① 難易度ごとのパラメータ管理
弾数・弾速・当たり判定サイズをオブジェクトで管理し、
難易度によって動的に変化させる仕組みを実装しました。

### ② 状態管理の整理
ゲームの進行状態（プレイ中 / ゲームオーバー / メニュー）を
明確に分離し、可読性を意識しました。

### ③ UIとゲームロジックの分離
描画処理とロジック処理を可能な限り分け、
保守性を意識した構造にしています。

### ④ スマホ対応
タッチ操作UIを追加し、PC・スマホ両対応にしました。(現在修正中)
